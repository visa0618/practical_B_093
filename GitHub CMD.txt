
=> First time push:-

git init
git remote add origin https://github.com/visa0618/practical_B_093.git
git add .
git commit -m "Added"
git branch -M main
git push -u origin main



=> If Repository Already Exists (Second Time Push):-

cd <file location>
git add .
git commit -m "update"
git push


=> If You Are Using Someone Else's PC:-

cd <file location>
git clone https://github.com/visa0618/practical_B_093.git

# After this will create a new folder name practical_B_093

cd practical_B_093

#Go inside folder and copy paste the file which we are pushing to git
#For Eg: Move stud.txt into the cloned folder
 
git add .
git commit -m "Done"
git push
